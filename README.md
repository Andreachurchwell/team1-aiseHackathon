# Our Door

Team Three's Company hackathon/capstone MVP.

## What This Repo Has
- `index.html` - 7-slide pitch deck for Our Door.
- `styles.css` - pitch deck styling.
- `our-door-app.html` - React frontend prototype for Our Door.
- `our-door.css` - app prototype styling.
- `pitch_script.md` - slide-by-slide speaking script.
- `SPEAKING_POINTS.md` - shorter speaking notes.
- `MVP_FEATURES.md` - MVP in/out scope.
- `ARCHITECTURE_SKETCH.md` - system architecture notes.
- `JOURNEY_MAP.md` - student/admin journey notes.

## Our Door Concept
Our Door is a curriculum-aware doorway into understanding. It gives students one safe place to ask questions from the program's own materials, and gives instructors visibility into repeated confusion before students fall behind.

## How to Open
- Pitch deck: open `index.html`.
- Frontend prototype: open `our-door-app.html`.

The prototype uses React from a CDN, so the browser needs internet access for the React scripts to load.
