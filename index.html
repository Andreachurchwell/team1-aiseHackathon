Two-Role Learning Assistant

Hackathon / capstone MVP pitch

Problem

Deliverable 1

Problem Framing

Cohort programs move fast and have a lot of materials: PDFs, slides, markdown files, recordings, notes, and assignments.

Students Get overwhelmed and can feel behind quickly.
Current fallback Ask classmates, Google, or use ChatGPT for direct answers.
Our angle Use the program's own materials and show repeated confusion before students fall behind.
Say: The problem is not just access to answers. The problem is learning support and visibility.

Deliverable 1

Users + Why It Matters

Student

Asks questions in chat and receives Socratic guidance grounded in the actual curriculum.

Admin / Instructor

Sees logged question themes and stuck points so support can improve without technical help.

The goal is safe learning help for students and clear signals for admins.

Say: Students need help without answer dumping. Admins need patterns without needing to inspect the system manually.

Deliverable 3

MVP Scope by June 2

IN

  • Canvas corpus ingested and queryable with real content
  • Backend ingestion path for admin-provided course coverage materials
  • Student chat UI
  • Socratic response instead of direct answers
  • Guardrails on every response
  • Conversation logging: question, response, timestamp, topic
  • Admin view with logged conversations
  • Two hardcoded roles: student/admin
  • Docker Compose + GitHub CI

OUT

  • Individual user accounts or registration
  • Student-specific history
  • Analytics or visualizations
  • Polished file upload UI for admins
  • Multi-cohort support
Say: This is intentionally narrow. We are proving the student learning loop and the admin visibility loop.

What This Proves

The MVP proves a two-sided learning loop: students get guided help, and admins see where support is needed.

For students Safer questions and better learning support.
For admins Clearer visibility into curriculum friction.
For the program Less guesswork, faster intervention, stronger outcomes.
Say: This is not trying to replace teaching. It helps students learn and helps instructors know where to focus.
